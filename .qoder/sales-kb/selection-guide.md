# 选型指南与方法论

## 选型前：7531客户分析

在做任何选型之前，先用7531框架了解客户全貌。

### 7个基本信息
1. 行业细分趋势 — 客户所在行业是上升还是下行
2. 横向竞争关系 — 客户在其行业中的地位
3. 上下游分析 — 客户的供应链关系
4. 企业产值与利润 — 判断预算能力
5. 经营策略 — 成本导向还是品质导向
6. 组织架构类型 — 直线型(<5000万)/树型(5000万-3亿)/矩阵型(>3亿)
7. 自动化现状 — 当前使用的品牌和产品

### 中型OEM（树型架构）打法
我们的核心客户群是树型架构的中型OEM（产值5000万-3亿），特征：
- 有部门分工但决策相对集中
- **核心策略：找关键人**（技术总监/研发主管），而非直接找老板
- 关键人通常是技术背景，用技术方案打动他，再由他内部推动
- 组织开始有采购流程但还不复杂，抓住关键人就能推进

### 5个推理点
1. 客户为什么要换/升级自动化系统？
2. 决策链上有哪些角色？各自关注什么？
3. 竞品在客户端的渗透程度？
4. 客户的真实预算区间？
5. 项目时间线和紧迫程度？

## 选型流程

### 需求收集清单

**应用层面：**
- [ ] 设备类型与工艺流程
- [ ] 控制轴数与运动类型（定位/同步/凸轮）
- [ ] I/O点数与类型
- [ ] 通信协议要求
- [ ] 安全等级要求
- [ ] 环境条件（温度/湿度/防护等级）

**商务层面：**
- [ ] 目标成本区间
- [ ] 竞品使用情况
- [ ] 批量大小（单机/批量）
- [ ] 交付周期
- [ ] 后续服务需求

### 价值量化：两提四节

每个选型方案都应量化客户能获得的价值：

| 维度 | 如何量化 | 话术示例 |
|------|---------|---------|
| 提良率 | 故障率降低XX%、废品率减少 | "S7-1500的内置诊断可将故障排查时间缩短60%" |
| 提效率 | 产能提升XX%、换型时间缩短 | "TIA Portal统一编程环境，开发效率提升30%以上" |
| 节材料 | 减少原材料浪费 | "高精度伺服控制降低切削废料" |
| 节能源 | 能耗降低XX% | "G120变频器在泵/风机应用可节能30-50%" |
| 节空间 | 设备/电柜体积缩小 | "S210单电缆方案减少50%的布线空间" |
| 节人员 | 减少操作/维护人员 | "远程诊断功能可减少现场巡检频次" |

## 常见选型场景速查

### 场景1：小型包装机（5轴以下）
- 控制器：S7-1200 G2（够用且性价比高）
- 伺服：S200（PN/PTI接口灵活）
- HMI：Unified 7寸/10寸精智面板
- 低压：追加3RV2电动机保护 + 3RT2接触器
- 竞品常见配置：汇川H5U + SV660N + IT7000

### 场景2：中型设备（5-16轴运动控制）
- 控制器：S7-1500T（工艺对象强大）
- 伺服：S200/S210（按精度要求选择）
- HMI：Unified 12寸/15寸
- 关键卖点：TIA Portal一站式开发、工艺对象降低编程复杂度

### 场景3：高端多轴设备（16轴以上）
- 控制器：S7-1500T
- 伺服：S120（多轴共直流母线，节能）
- 关键卖点：DCC图形化编程、Safety Integrated、虚拟调试
- 竞品常见配置：博世力士乐CtrlX + IndraDrive

### 场景4：通用变频应用
- 经济型：V20
- 标准型：G120C（紧凑一体式）
- 高端型：G120（模块化、多功能）
- 基础设施：G120X（IP55防护、泵/风机优化）
