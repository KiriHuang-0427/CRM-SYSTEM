# 周报模板

每周末生成周报，输出为Excel文件（5个Sheet），路径：`D:\知识库创建\06_客户追踪\周报\YYYY-WNN_销售周报.xlsx`

## Excel结构（5个Sheet）

### Sheet 1: 本周拜访

| 列名 | 说明 |
|------|------|
| 日期 | 拜访日期（如 04/21 周一） |
| 客户 | 公司全称 |
| 关键人 | 拜访对象姓名+职位 |
| 拜访目的 | 一句话说明为何去 |
| 关键收获 | 一句话概括核心信息 |
| 竞品动态 | 当天了解到的竞品信息 |
| 行动承诺 | 从客户那里拿到的具体承诺 |
| 负责人 | 谁负责跟进 |
| 截止日期 | 承诺的完成时间 |
| 信任五环评分 | 概念★/信息★/标准★/优势★/承诺★ |

样式：带筛选器、冻结表头、交替行底色。紧急行（如华章900系列）标红色底色。

### Sheet 2: 下周计划

| 列名 | 说明 |
|------|------|
| 优先级 | 最高/高/中/低 |
| 客户 | 目标公司 |
| 计划日期 | 具体日期或周次 |
| 目的 | 去干什么 |
| 需准备 | 要带什么材料 |
| 涉及产品 | 哪条产品线 |
| 负责人 | 谁去 |
| 状态 | 待办/进行中/已完成 |
| 备注 | 补充说明 |

样式：按优先级颜色区分（红=最高/橙=高/绿=中）。

### Sheet 3: 项目Pipeline

| 列名 | 说明 |
|------|------|
| 客户 | 所属公司 |
| 项目名称 | 具体机会名称 |
| 涉及产品 | 产品线 |
| 阶段 | 样机窗口期/报价推进/技术验证/执行中等 |
| 预计量级/金额 | 数量或金额 |
| 主要竞品 | 对手品牌 |
| 紧急度 | 极紧急/紧急/中/低 |
| 下一步 | 具体行动 |
| 截止日期 | 时间要求 |

样式：按紧急度颜色区分（红=极紧急/橙=紧急）。

### Sheet 4: 竞品动态

| 列名 | 说明 |
|------|------|
| 竞品品牌 | 汇川/台达/伟创/丹佛斯等 |
| 涉及客户 | 渗透到了哪家 |
| 竞品产品 | 具体型号或产品线 |
| 威胁程度 | 极高/高/中/低 |
| 具体情况 | 细节描述 |
| 应对策略 | 我们的打法 |

样式：按威胁程度颜色区分（红=极高/橙=高）。

### Sheet 5: 复盘笔记

自由文本区域，包含以下板块（用背景色区隔）：
- **方法论实战**：用了什么工具、在哪个场景、效果如何、下次改进
- **本周学习**：产品知识、行业洞察、读书进度
- **关键发现**：战略机会、风险警示、方法论亮点
- **AI伙伴反馈**：做得好的、需关注的、下周最重要的三件事

---

## 命名规范

`YYYY-WNN_销售周报.xlsx`

例如：`2026-W17_销售周报.xlsx`

---

## 生成依赖

Node.js + ExcelJS 库。生成前执行 `npm install exceljs`（全局已安装则秒过）。
