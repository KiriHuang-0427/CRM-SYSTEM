API 接口清单
方法 路径 说明 请求参数
GET /api/customers 获取客户列表 ?search=&sort=priority
GET /api/customers/:id 获取客户详情 URL参数 id
POST /api/customers 新增客户 Body: CustomerForm
PUT /api/customers/:id 修改客户 Body: CustomerForm
DELETE /api/customers/:id 删除客户 URL参数 id
GET /api/pipeline Pipeline汇总统计 无
GET /api/pipeline/:customerId 客户商机列表 URL参数 customerId
PUT /api/pipeline/:id/stage 切换Pipeline阶段 Body: { stage: number }
GET /api/todos 获取待办列表 ?status=pending
POST /api/todos 新增待办 Body: { text, customerId?, deadline? }
PUT /api/todos/:id 更新待办状态 Body: { completed: boolean }
DELETE /api/todos/:id 删除待办 URL参数 id
GET /api/notes 获取笔记列表 ?customerId=
POST /api/notes 新增笔记 Body: { customerId, content }
DELETE /api/notes/:id 删除笔记 URL参数 id
